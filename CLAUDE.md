# Claude Code Instructions

## Versioning

- When bumping the version, always update `CHANGELOG.md` with the new entry before committing.
- Follow the [Keep a Changelog](https://keepachangelog.com/en/1.1.0/) format.
- Version is tracked in two places: `arbiter/package.json` and the MCP server info in `arbiter/server.ts`. Both must be updated together.
